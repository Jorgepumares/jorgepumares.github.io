Tipos de selectores que entran en examen

  1. Clase (class="")

    2. Los selectores de clase buscan un elemento basado en el contenido de su atributo class. El atributo class está definido como una lista de elementos separados por espacio, y uno de esos elementos debe coincidir exactamente con el nombre de clase dado en el selector.
    Ejemplo: Acabo de ver a Rafa Nadal ganar con España

  2. Identificador (id="")

    3. Los selectores de ID de CSS buscan un elemento basado en el contenido del atributo id. El atributo ID del elemento seleccionado debe coincidir exactamente con el valor dado en el selector.
    Ejemplo: Acabo de ver a Pedro Sanchez salir del Falcón

  3. Todos los elementos (*)

    4. El selector universal funciona como un carácter comodín, seleccionando todos los elementos en una página. Cada página HTML se basa en contenido colocado dentro de etiquetas HTML. Cada conjunto de etiquetas representa un elemento en la página.
    Por ejemplo: * se aplicará el estilo que elijas a todos los elementos

  4. Varios elementos ( p, em, h1, h2)

    5. Este atributo sirve paraa aplicar una mismas modificaciones a distintos tipos de texto.
    Ejemplo: Me acabo de comer un buen estofado

  5. Elementos contenidos en otros elementos (ol li)

    6. Selecciona todos los elementos de la página cuya etiqueta HTML coincide con el valor del selector. Para utilizar este selector, solamente es necesario indicar el nombre de una etiqueta HTML (sin los caracteres <>) correspondiente a los elementos que se quieren seleccionar.
    Ejemplo:

    1. Texto 1
    2. Texto 2

  6. Sobrevuleo (p:hover)

    7. Selecciona todos los elementos de la página cuya etiqueta HTML coincide con el valor del selector. Para utilizar este selector, solamente es necesario indicar el nombre de una etiqueta HTML (sin los caracteres <>) correspondiente a los elementos que se quieren seleccionar.
    Ejemplo: He seleccionado todos las etiquetas p, por los cual, cuando pases el raton por encima se pondrán de color amarillo.
